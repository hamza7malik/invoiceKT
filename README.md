# Invoice Generator - Vite + React

A modern, professional invoice generator built with React and Vite. Designed as a foundation for a SaaS product.

## 🚀 Quick Start

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## 📁 Project Structure

```
src/
├── components/
│   ├── InvoiceSettings.jsx    # Settings panel (left)
│   └── InvoicePreview.jsx     # Invoice preview (right)
├── utils/
│   └── formatters.js          # Formatting utilities
├── App.jsx                     # Main component
├── App.css                     # Styles
└── main.jsx                    # Entry point
```

## ✨ Features

- Set hourly rate and add work items
- Auto-calculate hours and totals
- Optional payment details
- Print/PDF export (Ctrl+P)
- Responsive side-by-side layout
- Clean, minimal corporate design

## 🛠️ Commands

- `npm run dev` - Start dev server
- `npm run build` - Build for production
- `npm run preview` - Preview build
